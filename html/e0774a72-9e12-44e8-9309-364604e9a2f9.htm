Privacy and Security

KotakNet DotNet Library

With KotakNet, your information and code is 100% safe

KotakNet is a standalone library and installed in the user system. Every communication from KotakNet happens directly from user computer to API server.

Requests are not routed through Howutrade or any 3rd party servers. Howutrade or any other don't have access to your system or code.

KotakNet is 100% free from malware, adware, spyware, viruses and trojans.

Always download KotakNet from Howutrade's official Github page or Website and verify the digital signature and certificate on the Binary files

Why routing order through 3rd party servers is evil?

  • Your order details are exposed

  • Any resource limitation at 3rd party server will impact your order (may fail or delay or result in slippages)

  • If the 3rd party server down, your order request will also fail (if it's a stop order, imagine the worst)

 Security Note

Keep the system and Antivirus upto date. Do not visit malicious websites. Do not download and install softwares from untrusted sources
